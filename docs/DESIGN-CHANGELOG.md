# 设计变更记录

## 2026-05-09 · v0.2 医疗科研工作台基线

背景：

- 用户要求优化 UI/UX，去除明显 AI 味、蓝紫渐变和模板感。
- 项目目标是医疗/科研场景，视觉需要更克制、专业、可信。

关键决策：

- 从深色玻璃拟态改为浅色医疗工作台。
- 主色改为低饱和青绿，配合暖灰/墨色中性色。
- 移除光晕、蓝紫渐变、渐变分数和英文装饰标签。
- 保留原生 HTML/CSS/ES Modules 技术栈，不引入新前端框架。
- 修复判读页备注输入连续编辑时的保存监听问题。

影响页面：

- 首页
- 登录/注册
- 判读页
- 结果页
- 研究图像集工作台
- 单任务图像管理
- 研究管理中心
